Split Manufacturing Order

Key Features:

  • Allows you to split manufacturing order.
  • User who have access rights can split manufacturing order.
  • User have different split options as like below.

    Number of Split: It will divide quantity into selected number of split.

    Number of Quantity: Each split MO quantity as selected number of quantity.

    Manual Split: User have to enter the quantity for each split.

Overview

  • From Users menu, there is access rights called Can Split Manufacturing Order.
  • From Manufacturing Orders menu, there is button for split manufacturing order. Click on button for split MO.
  • In pop-up, there is option called split MO by. Here we select the option Number of Split. So it will create number of split MO based on number entered.
  • After Split, see below screenshot. It generate MO based on entered number in wizard.
  • If user select the option Number of Quantity, then user have to enter the how many quantity in each split MO.
  • After split, the number of quantity entered by user, that much quantity in each split MO.
  • If user select option Manual Split, then user have to define how many quantity for each split MO.
  • After split, based on the user input, the quantity will be assign on each split MO.
